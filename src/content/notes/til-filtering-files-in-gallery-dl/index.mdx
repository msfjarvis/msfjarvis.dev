You can filter for things like file names using the \`--filter\` option in [gallery-dl](https://codeberg.org/mikf/gallery-dl) which takes a Python expression. \`gallery-dl <URL> --filter "'Thing' in filename" is a valid filter, but I couldn't find a definitive resource for all supported fields. I saw mention of \`image_width\` in a help text so that's one, and you can probably land on the one you want by guesstimating.
