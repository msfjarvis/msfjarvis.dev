I'm an Android and Kotlin Multiplatform engineer with 9 years of experience building apps and SDKs that have scaled up to hundreds of thousands daily active users for some of the largest startups in India.

# Work

I currently work at [Cloudflare](https://cloudflare.com) as a Systems Engineer. I was previously at [Dyte](https://dyte.io), which was acquired by Cloudflare in early 2025.

You can find the latest version of [my résumé here](/resume.pdf).

# Open Source

Some of my open source projects:

- [Android Password Store](https://passwordstore.app): A password manager for Android aiming to be fully compatible with the [pass](https://passwordstore.org) format. I took over maintainership from its original author and maintained it for nearly 6 years before stepping down.
- [Claw](https://msfjarvis.dev/g/compose-lobsters): An Android client for [lobste.rs](https://lobste.rs), written entirely in [Jetpack Compose](https://developer.android.com/jetpack/compose/). This is also my experimentation test bed where I learn new libraries and frameworks in a practical setting.
- [healthchecks-rs](https://msfjarvis.dev/g/healthchecks-rs): A Rust library for interacting with [healthchecks.io](https://healthchecks.io) and a couple of CLI tools that build on the library.
- [linkleaner](https://github.com/msfjarvis/linkleaner): Telegram bot to automatically improve link previews.
